STIM Light Version - Model And Data Summary
Generated on: 2026-05-12

1) Goal of the light setup
- Run a smaller but protocol-faithful transfer pipeline:
  1. Tune one config per regime with Optuna on source family.
  2. Rerank top trials with raw Monte Carlo on held-out source graphs.
  3. Freeze winner config per regime.
  4. Evaluate transfer on larger target graphs.

2) Model overview (same core STIM architecture)
- Diffusion surrogate:
  - Progressive temporal GNN diffusor.
  - Predicts activation rollout from seed choices and temporal edge weights.
- Policy:
  - DQN-style sequential seed selector.
  - Uses candidate features from activation state, temporal summaries, and spatial features.
- Regimes:
  - spread: objective mostly activated mass.
  - dynamic: temporal sequence objective.
  - spatial: activated mass + coverage - cost.
